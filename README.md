# 🏠 Dwellyo.ai - AI-Powered Rental Management

The most advanced AI platform for property management. Automate tenant communication, viewings, screening, and contracts with intelligent automation.

## ✨ Features

- 🤖 **Intelligent AI Responses** - 24/7 tenant communication
- 📅 **Smart Scheduling** - Automated viewing appointments  
- 👤 **AI Tenant Screening** - Advanced background checks
- 📋 **Digital Contracts** - Automated lease management
- 📊 **Analytics Dashboard** - Performance insights
- 🔧 **Maintenance Management** - Issue tracking & resolution

## 🚀 Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: Supabase (PostgreSQL)
- **Frontend**: EJS Templates, Modern CSS
- **AI**: OpenAI GPT Integration
- **Authentication**: Session-based auth with bcrypt
